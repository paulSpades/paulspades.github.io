The Purely Bermuda website is a part of a family of destination-centric travel websites by Purely Travel UK. They've partnered with Trip Solution for many years to develop their online business.

This design was created and first implemented in 2013 using our new CMS software, but re-done in 2015 to take advantage of Bootstrap 3's new responsive grid system.

One of my challenges was re-coding the slideshow from the header so that it replicates the flash based one (not visible in the screen capture) on their old websites. I reiterated in Javascript until I came to a simple, readable functional programming style solution.

This project was completed while I was employed at Trip Solutions.
